# Heavy_Note

  Esse programa tem a função de criar um arquivo de tamanho personalizável, útil apenas para ocupar espaço no seu HD durante teste de TI ou a fim de te dar uma desculpa para não baixar aqueles jogos aleátorios que seus amigos pedem =)
  
  * Execução Windows:<br>
    Após realizar a instalação do python, basta clicar com o botão direito no main.py e executar como administrador.
    <br><br>
  * Execução Linux:<br>
    No terminal, vá até a pasta onde está o main.py e execute o comando "sudo python3 main.py"
